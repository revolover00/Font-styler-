## ◉ QUICK START

# Clone
git clone https://github.com/revolover00/Font-styler-.git

# Install
npm install

# Set Gemini API key
echo "GEMINI_API_KEY=your_key_here" > .env.local

# Run
npm run dev

# Build
npm run build
